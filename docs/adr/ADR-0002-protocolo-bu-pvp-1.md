# ADR-0002: Adopción de BU-PVP-1

## Estado

Adoptado

## Contexto

Se requiere voto secreto con escrutinio verificable públicamente y evidencia anclada.

## Decisión

Adoptar BU-PVP-1: boletas cifradas publicadas + procesamiento/tally con pruebas verificadas on-chain + actas firmadas/ancladas.

## Consecuencias

- Complejidad criptográfica y operacional elevada.
- Verificabilidad pública fuerte si contratos/circuitos son correctos.
