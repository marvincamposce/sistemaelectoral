# ADR-0003: Modelo de confianza electoral

## Estado

Adoptado

## Contexto

Un sistema electoral requiere separación entre autoridad, registro, recepción, escrutinio, observación y auditoría.

## Decisión

Definir actores: AEA, REA, MRD, JED, Observador, Auditor, API, Indexer, Smart contracts.

## Consecuencias

- Se reduce el impacto de compromisos parciales.
- Se habilita detección pública de comportamientos maliciosos.
